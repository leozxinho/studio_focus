Sprint para planos da academia:

Plano mensal:

Valor: 50$
Benefícios: suporte básicos do instrutor, treino inicial

Plano semestral:
Valor: X$
Benefícios: Auxílio mais próximo durante o treino, Treino personalizado, 1 avaliação física a cada 3 meses, Reavaliação de progresso (peso + medidas), Ajustes no treino conforme evolução.


Plano anual:
Valor: X$
Benefícios:  Treino completo personalizado (com atualização frequente), Acompanhamento mais próximo do instrutor, Avaliação física mensal, Análise de percentual de gordura, Cálculo de IMC + Relatório técnico, Orientação alimentar, Desafios mensais.
